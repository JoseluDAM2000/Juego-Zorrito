# Juego-Zorrito

Un juego simple para uno o dos jugadores en el cual tendrás que reunir las gemas que sueltan los enemigos al morir y suficiente comida para pasar el nivel.

## Controles

* Con las flechas de direccion moveremos al zorrito por el escenario para que pueda recojer la comida y los diamantes que encuentre.
* Con la tecla Shift nuestro zorrito correra como alma que lleva el diablo para recojer los objetos que se encuentren cerca de desaparecer.
* Con el Click izquierdo del ratón eliminaremos a los aguilas.
